h1. AWS User Group Helsinki

The site at http://www.aws-helsinki.fi

h2. Setup

* Install 's3cmd' (a new version from sources is likely to work, brew/port might not)
* Setup S3 credentials (ask Matti at Enemy)
** @s3cmd --configure@
** (Enter credentials)
* @bundle install@

h2. Deploy

* @bundle exec jekyll@
* @cd _site@
* @../deploy.sh@ (do *not* run at 'application root', remember to cd to '_site' first)
* Check "http://aws-helsinki.fi.s3.amazonaws.com/index.html":http://aws-helsinki.fi.s3.amazonaws.com/index.html
* Wait up to 24 hours to be displayed at "http://aws-helsinki.fi/":http://aws-helsinki.fi/
